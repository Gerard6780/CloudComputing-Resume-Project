# Mis Proyectos

## Portfolio Serverless en AWS

**Tecnologías:** AWS Lambda, DynamoDB, API Gateway, Amplify, Terraform, Python, Hugo

Desarrollo de un portfolio profesional completamente serverless utilizando servicios de AWS:

- **Frontend**: Sitio estático generado con Hugo y desplegado en AWS Amplify
- **Backend**: Función Lambda en Python que gestiona datos del CV
- **Base de Datos**: DynamoDB para almacenamiento de información
- **API**: API Gateway REST para comunicación frontend-backend
- **Infraestructura**: Terraform para IaC (Infrastructure as Code)
- **CI/CD**: GitHub Actions para testing y validación automática

### Características Destacadas

✅ Arquitectura completamente serverless  
✅ Escalabilidad automática  
✅ Bajo costo operativo  
✅ Despliegue automatizado  
✅ Testing automatizado con pytest  
✅ Infraestructura versionada con Terraform
